# Music Folder

Place your wedding song MP3 file here as:

  public/music/wedding-song.mp3

The file path is configured in:

  src/data/weddingData.js  →  music.src

You can use any royalty-free MP3. Recommended free sources:
- https://pixabay.com/music/ (search "wedding" or "classical")
- https://freemusicarchive.org/

The music player button will appear in the bottom-right corner.
Music is disabled by default — the guest taps the button to play.
